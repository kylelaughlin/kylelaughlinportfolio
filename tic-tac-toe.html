As an exercise in writting vanilla javascript, I wrote a game of tic-tac-toe complete with score counter and refresh button. No page refreshes necessariy. This game utilizes event listeners on each board tile to trigger a function that handles the players selection, checks for a winner, and progresses the game appropriately(next move, player wins, tie game).

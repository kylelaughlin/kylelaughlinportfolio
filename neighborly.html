Neighborly

Neighborly is a web app that allows a community to trade time and services without the exchange of money. Ask for help and your neighbors can make offers to lend a hand. This application was an opportunity to learn about user authentication and authorization, ruby on rails, Bootstrap with custom CSS, and image uploading while utilizing Amazon Web Services S3 for remote storage.
